HTML İLE NELER YAPILABİLİR?

HTML ile web sayfaları tasarlayabilirsiniz. son zamanlarda gelişen teknolijilerle birlikte

HTML'i mobil uygulamaa-larda da kullanılmak mümkündür. Tüm internet siteleri HTML'den oluşmaktadır. Tabi ki tek başına HTML bir şey ifade etmez.

Web sayfalarında kullanıcıların gördüğü bütün kısımlar HTML ile yapılmkatadır. Yazılar, resimler, başlıklar ve diğer şeyler HTML ile yapılır.

Ancak yukarıda da belirtiğim gibi sade HTML size düz ve tasarımsız bir sayfa sunacaktır.

HTML - CSS - JAVASCRİPT

HTML ile sayfaların genel olarak iskeletini çıkartırsınız. Sıralamaları, pozisyonları, içeriklerini ayarlarsınız. Tasarım kısmında CSS yardımcı olmaktadır.

CSS ile projelerin tasarımları gerçekleştirilir. Arkplan renginden, yazı fontuna kadar her şey CSS ile yapılır.

Javascript ise oluşturduğunuz projelerin animasyonlarını sağlar. CSS ile animasyon verebilirsiniz ancak Javascript kadar etkili olmayacaktır. Son zamanlarda Javascript animasyon dışında çok farklı şeylerde de kullanılmaya başlandı.

En basitinden kullanıcının tıkladığı kısımda aksiyonu Javascript ile gerçekleştirirsiniz.Misal olarak ekranda açılan Pop-up kutusunu çarpı tuşuna basıp kapatma işlemi, Javascript tarafından sağlanır

Genel olarak bu üç dil için örnek vermek gerekirse: İnsan vücudunda, organların varlığını HTML, organların görünüşünü CSS, hareketlerini ise Javascript ile sağlarız. Back-end dilleri ise organların Çalışmasını sağlar.

HTML NE KADAR SÜREDE ÖĞRENİLİR?

İyi bir takipçi ve çalışkan bir öğrenciyseniz, ortalama 3-4 günde HTML ve CSS'in teori kısmını tamamlayabilirsiniz. Geri kalan zamanda da pratikler yapıp, bu konuda gelişmeye başlayabilirsiniz.

Ortalama bir kişi ise bir hafta ila 15 gün arasında bir sürede HTML ve CSS'in teori kısmını tamamlayabilir.

HTML ve CSS bir programlama dili olmadığı için, öğrenmesi basittir. Programlama dillerinde ki gibi algoritmalar yoktur. Yazar ve Şekillendirirsiniz.

HTML NE KADAR GEREKLİ?

HTML, web projeleri gerçekleştirmek isteyen insanlar için son derece şarttır. Back-end alanında çalışıyor olsa bile biraz HTML, biraz CSS, biraz Javascript Bilmek zorundadır.

Sadece HTML öğrenerek bir şey yapamazsınız. yazıda da belirtiğimiz gibi sadece HTML bir şey ifade etmez. Yanında CSS ve Javascript öğrenmeniz gerekecektir.

Web projeleri alanında yürümek istiyorsanız, kesinlikle öğrenmeniz gerekiyor. Teorisini öğrendikten sonra, pratiklerle kendinizi geliştirmelisiniz. Youtube üzerinde ücretsiz şekilde öğrenebilirsiniz.
