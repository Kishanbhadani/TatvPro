CSS NEDİR?

Nedir bu CSS? Ne işe yarar? CESESE diyemi okunur? Silin kafanızda ki soruları. Buraya odaklanın lütfen!

CSS'in açılımı "Cascading Style Sheets" demektir. Yani "Basamaklı Biçim Sayfaları" demektir. "Ee yani? "diyenleriniz olacak. Okumaya devam o zaman.

Bir önce ki dersimizde HTML'den bahsetmiştik. HTML'nin tek başına bir şey ifade etmediğini, diğer dillerinde de yardım alınması gerektiğini söyledik.

CSS, HTML'i biçimlendirmenizi sağlayan tasarım dilidir. HTML ile içerikleri hazırladığınız projelerinizin görünümünü CSS ile şekillendirebilirsiniz.

Yani elinizde bir oyun hamururu aldığınızı düşünün, bu hamurun rengini, şeklini,duruşunu,hareketlerini CSS ile yapabilirsiniz. HTML bir Hamur, CSS ise verdiğiniz niteliklerdir.

Misal olarak HTML'de bir yazı yazdınız. BU yazının büyüklüğünü, rengini,kalınlığını CSS ile düzenleyebilirsiniz.

CSS NASIL KULLANILIR?

CSS,HTML'de üç farklı şekilde kullanılabilir.Son zamanlarda Javascript ile de kullanılmaya başlandı. Ama ben eski kafadan size anlatacağım.

CSS'i, HTML tagları içerisinde "style" attribute ekleyebilirsiniz, HTML taglarının arasında "style" tagı açarak kullanılabilirsiniz veya extra dosya olarak sayfanıza include edebilirsiniz.

En temiz yöntem include etmek yöntemidir. İnclude yaptığınızda CSS, HTML'den ayrılacağı için temiz kod yazmanıza olanak sağlar.

CSS NE KADAR SÜREDE ÖĞRENİLİR

CSS'in teorik kısmını ortalama 5 gün ile 15 gün arasında tamamlayabilir. Tabi ki bu süreç, kişiden kişiye göre değişir. Çok istiyorsanız, bencede 5 günde bitirirsiniz. Ufaktan ingilizceniz varsa daha rahat edeceksiniz.

Eğer ağır öğreniyim gibi bir düşünceniz varsa bence yazılıma devam etmeyin. Yavaşça ilerleyebildiğiniz işler kovalayın.

Yazılım sürekli tekrar gerektiren, pratik gerektiren bir oluşum. Tekrarlamadığınız şeyleri kısa zamanda unutabilirsiniz.

Teori kısmını geçtikten sonra, işin zevkli kısmı başlıyor. Deneyerek, daha güzel tasarımlar çıkarmaya başlayacaksınız!

En basitinden kullanıcının tıkladığı kısımda aksiyonu Javascript ile gerçekleştirirsiniz.Misal olarak ekranda açılan Pop-up kutusunu çarpı tuşuna basıp kapatma işlemi, Javascript tarafından sağlanır

Genel olarak bu üç dil için örnek vermek gerekirse: İnsan vücudunda, organların varlığını HTML, organların görünüşünü CSS, hareketlerini ise Javascript ile sağlarız. Back-end dilleri ise organların Çalışmasını sağlar.

CSS NE KADAR GEREKLİ?

Düşünün ki bir tasarım ödevi yapacaksınız. siz A4 kağıdına düz yazı yazıp gittiniz ve sıfır almanız çok olağan. HTML sizin yazdığınız düz yazının halidir. CSS isee bu yazının şekillenmesini sağlar.

CSS, tasarımın bel kemiğidir. Göürünen kısmında ki bütün görsel değişiklikler CSS ile yapılır. Front-End çalışan kişilerin kesinlikle bilmesi gerekir. Back-End için çok gereklimidir tartışılır. Bence Back-End için çok gerekli bir şey değil.

Sizde HTML'de olduğu gibi web ve mobil alanında heves duyuyorsanız, kesinlikle öğrenmelisiniz.
